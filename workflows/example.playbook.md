# Example playbook

An ordered list of Tasks Aria runs end-to-end when you say "run the example
playbook". Each numbered step below becomes one durable Task; she works through
them IN ORDER, pings you as each finishes, and stops to ask only if one hits a
wall. Walk away — check progress anytime with "how are my tasks?".

This file IS the behavior: edit the steps, or add your own
`workflows/<name>.playbook.md`, and the playbook changes — no code edit needed.
Only the numbered/bulleted list items are run; this prose is ignored.

1. Check my unread email and give me a one-line summary of each.
2. Tell me what's on my calendar for today.
3. List any Cursor threads that are waiting on a decision right now.
